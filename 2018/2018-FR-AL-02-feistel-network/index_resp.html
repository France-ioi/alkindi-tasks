Réseau de Feistel

VOTRE MISSION

Cliquez sur les flèches bleues pour modifier les valeurs dans le but d'obtenir le nombre cible dans la case verte tout en bas du réseau.

Résultat à atteindre :

Le réseau fonctionne avec les opérations suivantes :

  • l'opération permute l'ordre des chiffres, par exemple 123 devient 312 ;

  • l'opération est une addition chiffre à chiffre (attention, il n'y a pas de retenues).
